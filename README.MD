## Structure

1. /all

Unsorted + unmodified versions of UrQMD files

2. /base

Unmodified versions of UrQMD files

3. /modified

Modified

4. Idea

At the end of the work there should be NO files in /all. Unmodified versions should be copied and stored in /base. 

If we DO NOT need to modify that file, there no need to add that file to /modified
If we NEED to modify, copy file to both /base and /modified

Empty /all means that work is completed.
